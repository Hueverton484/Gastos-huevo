Conectar tu spreadsheet personal
Pegá acá la URL del Apps Script que desplegaste para cargar datos de Salud/Gym/Comidas. Termina en /exec.
¿Cómo obtengo esa URL? (5 pasos)
  1. Abrí tu spreadsheet personal en una pestaña
  2. Menú Extensiones → Apps Script
  3. Borrá el contenido y pegá el código del archivo Personal-AppsScript.gs que está en la carpeta del proyecto
  4. Guardá (Ctrl+S) → Implementar → Nueva implementación → tipo "Aplicación web" → Ejecutar como "Yo" / Acceso "Cualquier persona"
  5. Copiá la URL que termina en /exec y pegala arriba
